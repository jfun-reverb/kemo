close
상태 의미 클라이언트(인플루언서) 노출 화면 표시
draft 작성·검토중 노출 안 됨 —
scheduled 모집 시작 전 노출됨 「모집예정」 / 「近日公開」 배지
active 모집중 노출됨 「모집중」 / 「募集中」 배지
paused 일시정지 노출 안 됨 —
closed 모집 종료 캠페인 노출 마감일까지 노출 「募集締切」 오버레이 + 신청 버튼 비활성
자동 전이 규칙
  • 모집 마감일(deadline) 경과하면 active → closed 로 자동 변경됩니다 (인플루언서 페이지 접속 시 클라이언트 측에서 처리).
  • 모집 마감일이 지난 캠페인은 active / scheduled 상태로 저장·변경할 수 없습니다.
  • closed 라도 캠페인 노출 마감일(post_deadline)이 남아 있으면 사이트에서 보입니다 (이미 응모한 인플루언서가 캠페인 정보·결과물 제출 페이지를 열 수 있도록).
전이 흐름 예시
draft → scheduled → active → closed → (post_deadline 경과 후 비노출)
paused 는 임시 중단용 — 작업 끝나면 active 로 복귀
