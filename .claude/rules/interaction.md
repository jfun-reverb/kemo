# 상호작용 규칙

## 선택지 질문
- 구현 방향, 옵션, 설정 등 선택이 필요한 질문은 AskUserQuestion 도구를 사용하여 키보드로 선택 가능하게 제공
- 한 번에 1개 질문씩, 순차적으로 진행
- 추천 옵션이 있으면 "(Recommended)" 표기

## 변경 프로세스
- 개선 요청 시 바로 구현하지 말고 계획 먼저 설명
- 관리자 페이지나 다른 기능에 영향이 있는지 체크 후 알림
- 애매한 부분이 있으면 반드시 먼저 질문

## 규칙/커맨드 기록
- 새로운 규칙이나 작업 패턴이 확인되면 항상 .claude/rules/ 또는 .claude/commands/에 기록
- 기존 규칙 파일이 있으면 업데이트, 없으면 새 파일 생성
- 사용자가 "기억해줘", "규칙으로 만들어줘" 요청 시 즉시 반영
