# 응모건 메시지 자동응답 — FAQ 실제 답변 문안 (한/일 전문)

> **작성일:** 2026-05-21
> **작성:** 기획/설계 세션
> **상태:** 답변 원문 1차 확정 (운영 검토 후 자동응답 등록)
> **목적:** 인플루언서 문의에 **실제로 보낼 일본어 답변**과 **관리자 등록·이해용 한국어 답변**을 질문별로 확정. 자동응답(FAQ) 콘텐츠에 그대로 등록하거나, 관리자가 LINE·메시지로 직접 답변할 때 복사해서 쓴다.
> **상위 사양:** [`docs/specs/2026-05-21-message-faq.md`](../specs/2026-05-21-message-faq.md) (부록 A의 답변 본문 = 이 문서)

## 사용 설명 (관리자용)

각 질문은 아래 6칸으로 구성됩니다.

| 칸 | 의미 |
|---|---|
| **質問 (일본어 제목)** | 인플루언서가 선택지 트리에서 보는 질문 제목 |
| **回答 (일본어 답변)** | 인플루언서에게 실제로 보내는 답변 본문 |
| **질문 (한국어)** | 관리자가 등록·관리할 때 이해용 |
| **답변 (한국어)** | 일본어 답변의 의미. 관리자가 LINE 등에서 한국어로 사고할 때 참고 |
| **화면 이동** | 답변 카드에 붙이는 앱 화면 바로가기 버튼 (없으면 「-」) |
| **사람 연결 / 단계** | `handoff`(바로 직접 문의) 여부 + 맞춤 노출 단계 태그 |

**일본어 작성 원칙 (사양서 §7 톤 가이드)**
- 정중하되 간결한 です・ます체. 과한 경어·딱딱함 지양
- 이모지·아이콘 금지 (텍스트만)
- 부정적 상황(반려·비선정)은 완곡하게, 다음 행동을 명확히
- 화면 이름은 앱 실제 일본어 라벨 그대로: 応募履歴 / 基本情報 / SNSアカウント / 配送先 / PayPal / パスワード変更 / メール受信設定 / 成果物提出
- 한국어 답변과 의미·길이를 맞춤

> **표기 주의:** 일본어 답변 안의 「マイページ」「応募履歴」 등 화면 이름은 **앱 UI 라벨과 정확히 일치**해야 인플루언서가 헤매지 않습니다. UI 라벨이 바뀌면 이 문안도 함께 수정해야 합니다.

## 카테고리 1. 申し込み・応募 (신청·응모)

### Q1-1. 応募ができません (신청이 안 돼요)

> 이 질문은 원인이 5가지라 **하위 선택지**로 나눕니다 (트리 2단). 아래 5개가 Q1-1의 자식 항목입니다.

#### Q1-1-a. メール認証が必要と表示されます

- **回答 (일본어):**
  「ご応募にはメールアドレスの認証が必要です。ご登録のメールアドレスに届いている認証メールのリンクを開いてください。メールが見つからない場合は、迷惑メールフォルダもご確認ください。それでも届いていない場合は、下の「直接お問い合わせ」からご連絡ください。」
- **질문 (한국어):** 이메일 인증이 필요하다고 떠요
- **답변 (한국어):** 응모하려면 이메일 인증이 필요합니다. 가입 시 받은 인증 메일의 링크를 열어주세요. 메일이 안 보이면 스팸함도 확인하시고, 그래도 없으면 직접 문의 바랍니다.
- **화면 이동:** -
- **사람 연결 / 단계:** 메일 미수신 시 handoff 안내 / 단계 무관

#### Q1-1-b. 必須情報を入力してくださいと表示されます

- **回答 (일본어):**
  「ご応募の前に、マイページで以下の情報のご登録が必要です。
  ・SNSアカウント（応募するキャンペーンの対象チャネル）
  ・配送先（郵便番号・都道府県・市区町村・電話番号）
  ・PayPalメールアドレス
  未登録の項目には「未登録」と表示されます。下のボタンからマイページを開いてご登録ください。」
- **질문 (한국어):** 필수 정보를 입력하라고 떠요
- **답변 (한국어):** 응모 전 마이페이지에서 ①SNS 계정(응모할 캠페인 대상 채널) ②배송지(우편번호·도도부현·시·전화) ③PayPal 이메일을 등록해야 합니다. 미등록 항목은 「未登録」으로 표시됩니다.
- **화면 이동:** 基本情報 (기본정보)
- **사람 연결 / 단계:** - / 단계 무관

#### Q1-1-c. フォロワー数が足りないと表示されます

- **回答 (일본어):**
  「このキャンペーンには最低フォロワー数の条件があります。応募対象として登録された代表SNSのフォロワー数が条件に達している必要があります。フォロワー数を最新の数値に更新してから、再度お試しください。最新化はマイページのSNSアカウントから行えます。」
- **질문 (한국어):** 팔로워 수가 부족하다고 떠요
- **답변 (한국어):** 이 캠페인은 최소 팔로워 수 조건이 있습니다. 응모 대상 대표 SNS의 팔로워 수가 조건을 충족해야 합니다. 팔로워 수를 최신값으로 갱신한 뒤 다시 시도해 주세요.
- **화면 이동:** SNSアカウント (SNS 등록)
- **사람 연결 / 단계:** - / 단계 무관

#### Q1-1-d. すでに応募済みと表示されます

- **回答 (일본어):**
  「同じキャンペーンには重複してご応募いただけません。すでにご応募済みの場合、応募状況は応募履歴からご確認いただけます。」
- **질문 (한국어):** 이미 신청했다고 떠요
- **답변 (한국어):** 같은 캠페인에 중복 응모는 불가합니다. 이미 응모한 경우 응모이력에서 상태를 확인할 수 있습니다.
- **화면 이동:** 応募履歴 (응모이력)
- **사람 연결 / 단계:** - / 단계 무관

#### Q1-1-e. 募集が締め切られたと表示されます

- **回答 (일본어):**
  「このキャンペーンは募集が締め切られています。募集人数に達した場合や募集期限を過ぎた場合は、新たなご応募を受け付けておりません。他のキャンペーンは応募履歴・キャンペーン一覧からご確認いただけます。」
- **질문 (한국어):** 모집이 마감됐다고 떠요
- **답변 (한국어):** 이 캠페인은 모집 마감입니다. 모집 인원이 찼거나 마감일이 지나면 신규 응모를 받지 않습니다. 다른 캠페인을 확인해 주세요.
- **화면 이동:** -
- **사람 연결 / 단계:** - / 단계 무관

### Q1-2. 応募を取り消したいです (응모를 취소하고 싶어요)

- **回答 (일본어):**
  「ご応募の取り消しは、応募履歴の各応募カードのメニューから行えます。なお、成果物がすでに承認されている場合など、取り消しができない場合があります。操作中にエラーが出る場合は、お手数ですが下の「直接お問い合わせ」からご連絡ください。」
- **질문 (한국어):** 응모를 취소하고 싶어요
- **답변 (한국어):** 응모 취소는 응모이력의 각 카드 메뉴에서 가능합니다. 단 결과물이 이미 승인된 경우 등에는 취소가 안 될 수 있습니다. 조작 중 에러가 나면 직접 문의 바랍니다.
- **화면 이동:** 応募履歴 (응모이력)
- **사람 연결 / 단계:** 에러 시 handoff / 단계 무관
- **⚠️ 운영 메모:** 실측에서 "취소가 에러로 안 됨" 사례 있었음. **취소 버그 수정 완료 전까지** 이 답변은 에러 시 handoff를 강조한 현재 문안 유지. 수정 후 안내 정확화.

### Q1-3. もう一度応募できますか (다시 신청할 수 있나요)

- **回答 (일본어):**
  「ご応募を取り消された後、募集が継続中であれば同じキャンペーンに再度ご応募いただけます。募集が締め切られている場合は、新たなご応募はできません。」
- **질문 (한국어):** 취소 후 다시 신청할 수 있나요
- **답변 (한국어):** 취소 후 모집이 진행 중이면 같은 캠페인에 재응모 가능합니다. 마감된 경우엔 불가합니다.
- **화면 이동:** -
- **사람 연결 / 단계:** - / 단계 무관

## 카테고리 2. 審査・結果 (심사·결과)

### Q2-1. 審査はいつ終わりますか (심사는 언제 끝나나요)

- **回答 (일본어):**
  「現在の審査状況は、このメッセージ画面の上部、または応募履歴でご確認いただけます。審査には数日いただく場合があります。結果が出ましたら、アプリの通知でお知らせします。今しばらくお待ちください。」
- **질문 (한국어):** 심사는 언제 끝나나요
- **답변 (한국어):** 현재 심사 상태는 메시지 화면 상단 또는 응모이력에서 확인할 수 있습니다. 심사에 며칠 걸릴 수 있고, 결과가 나오면 앱 알림으로 안내합니다.
- **화면 이동:** 応募履歴 (응모이력)
- **사람 연결 / 단계:** - / **단계: pending(심사중)** — 상태 한 줄로 1차 해소되므로 pending 응모건에서 우선 노출

### Q2-2. 当選したかどうか、どこで分かりますか (당첨됐는지 어떻게 아나요)

- **回答 (일본어):**
  「当選の結果は、応募履歴の各応募のステータスでご確認いただけます。当選された場合は「当選」と表示され、アプリの通知でもお知らせします。」
- **질문 (한국어):** 당첨됐는지 어디서 알 수 있나요
- **답변 (한국어):** 당첨 결과는 응모이력의 각 응모 상태에서 확인합니다. 당첨 시 「当選」으로 표시되고 앱 알림으로도 안내합니다.
- **화면 이동:** 応募履歴 (응모이력)
- **사람 연결 / 단계:** - / **단계: pending**

## 카테고리 3. 成果物の提出 (결과물 제출)

### Q3-1. レシート(領収書)はどう提出しますか (영수증을 어떻게 올리나요)

- **回答 (일본어):**
  「商品ご購入後、成果物提出の画面から領収書を提出してください。提出時には以下の3点が必須です。
  ・注文番号
  ・購入日
  ・購入金額
  あわせて領収書の画像をアップロードしてください。下のボタンから提出画面を開けます。」
- **질문 (한국어):** 영수증을 어떻게 올리나요
- **답변 (한국어):** 상품 구매 후 결과물 제출 화면에서 영수증을 제출합니다. 주문번호·구매일·구매금액 3가지가 필수이며, 영수증 이미지도 함께 올려주세요.
- **화면 이동:** 成果物提出 (활동관리/결과물 제출)
- **사람 연결 / 단계:** - / **단계: approved_purchase(구매기간 중)**

### Q3-2. 投稿URLの登録ができません (게시물 주소 등록이 안 돼요)

- **回答 (일본어):**
  「投稿URLを入力すると、チャネル（Instagram・TikTok・X・YouTubeなど）を自動で判別します。自動判別がうまくいかない場合は、チャネルを手動で選択してから提出してください。URLは投稿の公開ページのものをご入力ください。」
- **질문 (한국어):** 게시물 URL 등록이 안 돼요
- **답변 (한국어):** 게시물 URL을 넣으면 채널을 자동 인식합니다. 자동 인식이 안 되면 채널을 수동으로 선택한 뒤 제출하세요. URL은 게시물 공개 페이지 주소를 입력합니다.
- **화면 이동:** 成果物提出 (활동관리/결과물 제출)
- **사람 연결 / 단계:** - / **단계: approved_post(제출 단계)**

### Q3-3. 投稿の決まり(ハッシュタグ)はありますか (게시물 규칙이 있나요)

- **回答 (일본어):**
  「投稿の際は、ステルスマーケティング規制に準拠するため、#PR または #広告 などのタグを必ず表記してください。あわせて、ブランドが指定するハッシュタグ・アカウントがある場合は必ず含めてください。投稿は一定期間の掲載が必要です。詳しい条件は各キャンペーンのガイドラインをご確認ください。」
- **질문 (한국어):** 게시물 규칙(해시태그)이 있나요
- **답변 (한국어):** 게시 시 스텔스 마케팅 규제 준수를 위해 #PR 또는 #広告 태그를 반드시 표기하고, 브랜드 지정 해시태그·계정이 있으면 포함합니다. 게시물은 일정 기간 게시가 필요합니다. 자세한 조건은 각 캠페인 가이드라인 확인.
- **화면 이동:** -
- **사람 연결 / 단계:** - / 단계: approved_post

### Q3-4. 提出期限が過ぎました / 間に合いそうにありません (제출 기한이 지났어요 / 못 맞춰요)

- **回答 (일본어):**
  「提出期限を過ぎると、提出フォームが利用できなくなる場合があります。期限内の投稿が行われない場合、原稿料のお支払いができないことがあります。期限に間に合わない事情がある場合は、お早めに下の「直接お問い合わせ」からご相談ください。」
- **질문 (한국어):** 제출 기한이 지났어요 / 못 맞출 것 같아요
- **답변 (한국어):** 기한이 지나면 제출 폼이 비활성될 수 있습니다. 기한 내 미게시 시 원고료 지급이 안 될 수 있습니다. 사정이 있으면 가능한 빨리 직접 문의로 상담 바랍니다.
- **화면 이동:** -
- **사람 연결 / 단계:** **handoff (기한 상담은 사람 연결)** / 단계: approved_post

### Q3-5. 成果物が差し戻されました。再提出は? (결과물이 반려됐어요, 재제출은?)

- **回答 (일본어):**
  「成果物が差し戻された場合、成果物提出の画面上部に差し戻しの理由が表示されます。理由をご確認のうえ、内容を修正して再度ご提出ください。再提出すると、再び確認（審査）の状態に戻ります。」
- **질문 (한국어):** 결과물이 반려됐어요, 재제출은?
- **답변 (한국어):** 반려 시 결과물 제출 화면 상단에 반려 사유가 표시됩니다. 사유를 확인하고 내용을 수정해 재제출하세요. 재제출하면 다시 검수 상태로 돌아갑니다.
- **화면 이동:** 成果物提出 (활동관리/결과물 제출)
- **사람 연결 / 단계:** - / **단계: approved_post (반려 상태에서 우선 노출)**

## 카테고리 4. 配送 (배송)

### Q4-1. 商品はいつ届きますか (제품은 언제 오나요)

- **回答 (일본어):**
  「当選後、商品は順次発送されます。商品は韓国から発送されるため、お届けまでお時間をいただく場合があります。発送状況についてのご連絡があった場合は、アプリの通知やメッセージでお知らせします。一定期間を過ぎても届かない場合は、下の「直接お問い合わせ」からご連絡ください。」
- **질문 (한국어):** 제품은 언제 오나요
- **답변 (한국어):** 당첨 후 상품은 순차 발송됩니다. 한국에서 발송되어 도착까지 시간이 걸릴 수 있습니다. 일정 기간이 지나도 안 오면 직접 문의 바랍니다.
- **화면 이동:** -
- **사람 연결 / 단계:** 미도착 시 handoff 안내 / 단계: approved_purchase·approved_visit
