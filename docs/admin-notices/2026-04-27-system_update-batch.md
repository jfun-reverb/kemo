안녕하세요. 오늘(4월 27일) 오후 운영서버에 5가지가 업데이트됐습니다. 실무에 영향이 큰 순서로 정리했습니다.

**① 📅 캠페인 등록할 때 날짜 입력이 훨씬 편해졌습니다**

여러 개로 나뉘어 있던 날짜 칸을 묶었고, 새 항목도 추가됐습니다.

- 예전엔: "모집 마감일", "게시 마감일", "구매 시작일", "구매 종료일" 칸이 따로따로 있어 헷갈렸어요
- 지금은: 달력에서 **시작일 ~ 종료일을 한 번에** 끌어서 선택합니다
  - 모집 기간
  - 게시 기간
  - 구매·방문 기간
- **새 항목 — "모집 시작일"**: 캠페인이 인플루언서에게 언제부터 보일지 직접 정할 수 있습니다 (이전엔 등록 즉시 노출)
- **자동 변환**: 모집 시작일이 되면 "모집예정" 캠페인이 자동으로 "모집중"으로 바뀝니다
- **자동 추천**: 모집 종료일을 정하면 "결과물 제출 마감일"이 2주 뒤로 자동 추천됩니다 (수정 가능)
- **실수 방지**: 과거 날짜를 고르면 달력 안에서 바로 경고가 뜹니다
- **위치 정리**: 모집 타입(리뷰어/기프팅/방문)과 카테고리가 화면 위쪽 "기본 정보" 칸으로 이동해서 입력 순서가 자연스러워졌습니다
- **리뷰어 캠페인일 땐**: 콘텐츠 종류 선택지가 영상·이미지 위주로 자동으로 좁혀집니다 (Qoo10 리뷰는 글 콘텐츠가 없으니까요)

> **기존 캠페인은요?** 모집 시작일이 비어있는 기존 캠페인은 인플루언서 화면에서 종전대로 "오늘부터 마감일까지"로 보입니다. **새로 등록하는 캠페인부터** 모집 시작일을 채워주시면 됩니다.

**② 📌 공지사항 — 작성과 게시를 분리했습니다**

지금 보고 계신 이 공지도 새 흐름으로 올라가는 첫 공지입니다.

- 예전엔: 공지를 저장하는 순간 바로 모든 관리자에게 노출돼서, 미완성인 채로 실수로 나가는 사고 위험이 있었습니다
- 지금은: **"초안 저장"과 "게시하기" 두 단계**로 분리됐습니다

새 공지를 쓸 때 화면 아래 버튼:
- `[초안 저장]` — 본인과 최고관리자만 볼 수 있어요. 다른 관리자의 사이드바 배지·로그인 팝업·대시보드에 안 뜹니다
- `[게시하기]` — 모든 관리자에게 노출됩니다

이미 게시된 공지를 다시 수정할 때:
- `[게시 유지하며 저장]` — 노출된 채로 내용만 갱신
- `[초안으로 되돌리고 저장]` — 잠시 내려서 다듬기 (기본값 — 안전 우선)

공지 보기 화면에서:
- 작성자 본인과 최고관리자에게는 `[지금 게시]` / `[게시 회수]` 버튼이 보입니다
- 목록 위에 **"전체 / 게시 / 초안"** 필터가 추가됐습니다

> **참고**: 공지를 회수했다가 다시 게시해도 이미 읽은 사람의 미읽음 카운트는 다시 늘지 않습니다 (최초 게시 시점 기준).
