2026-05-07 운영 업데이트 안내

오늘(5월 7일) 관리자 페이지와 인플루언서 앱에 여러 개선 사항이 적용됐습니다. 운영하시면서 알아두실 내용을 정리했습니다.

관리자 페이지 — 캠페인 관리

캠페인 상태 「일시정지」가 사라지고 「노출마감」이 새로 생겼습니다. 이제 캠페인 상태는 5단계입니다.

  • 준비중 — 아직 모집을 시작하지 않은 캠페인 (회색 점선)
  • 모집예정 — 모집 시작일이 정해져 있고 시작 전 (파란색)
  • 모집중 — 지금 모집 중 (초록색)
  • 종료 — 모집은 끝났지만 게시 기한 안에 있어 인플루언서 화면에 보이는 캠페인 (분홍색)
  • 노출마감 — 게시 기한도 지나서 인플루언서 화면에 안 보이는 캠페인 (회색 점선)

「종료」에서 「노출마감」으로 시스템이 자동으로 바꿉니다. 게시 기한이 지나면 알아서 처리되니까 직접 안 바꾸셔도 됩니다.

캠페인 등록·편집 폼의 날짜 입력이 3줄로 정리됐습니다. 모집 기간 / 구매·방문 기간 / 결과물 제출 마감일을 좌우로 펼쳐서 한눈에 비교할 수 있습니다. 모집 종료일을 정하면 결과물 제출 마감일이 자동으로 +14일 뒤로 채워집니다.

「종료」 캠페인의 날짜를 수정해도 잘못된 차단 메시지가 더 이상 뜨지 않습니다.

관리자 페이지 — 결과물 관리

엑셀 다운로드가 더 보기 좋게 바뀌었습니다. 결과물 1개당 1줄이었던 것이 「신청 1개당 1줄」로 바뀌었습니다. 한 인플루언서가 낸 영수증·리뷰 이미지·게시물 주소를 한 줄에서 같이 볼 수 있습니다. 컬럼은 18칸 + 인플루언서 계정 이메일이 추가됐습니다.

결과물 검수 모달에서 리뷰 이미지도 같이 볼 수 있습니다. 영수증과 리뷰어 캠페인 2단계 리뷰 이미지를 한 화면에서 확인하면서 검수하시면 됩니다.

관리자 페이지 — 광고주 신청 (브랜드 서베이)

시딩 견적의 송금 수수료를 직접 입력하지 않으셔도 됩니다. 시딩 폼은 항상 0원으로 자동 채워집니다. 직접 등록 모달에서도 마찬가지입니다.

관리자 페이지 — 화면 공통

  • 표가 화면보다 길면 옆으로 끌어볼 수 있습니다. 캠페인 관리 / 신청 관리 / 결과물 관리 / 캠페인별 신청자 4개 페이지에서 노트북처럼 화면이 좁을 때 표를 좌우로 끌어 더 많은 컬럼을 보실 수 있습니다. 옆으로 밀어도 첫 번째 칸(주로 캠페인 썸네일·제목)은 왼쪽에 그대로 붙어 있어 어떤 행인지 헷갈리지 않습니다.
  • 필터 옵션 옆에 건수가 표시됩니다. 「(NN)건」 처럼 각 옵션별 건수가 보입니다. 결과물 페이지 필터에는 부가 설명 라벨도 추가됐습니다.
  • 수정·삭제 모달을 닫으면 뒷화면이 자동으로 새로고침됩니다. 방금 변경한 내용이 화면에 안 보이던 문제가 해결됐습니다.
  • 사이드바 메뉴를 클릭하면 페이지 전체가 깨끗하게 다시 뜹니다. 이전 화면에 떠 있던 모달이나 필터가 남아 헷갈리던 문제가 사라졌습니다.
  • 여러 항목을 고르는 드롭다운에서 긴 캠페인 제목이 자동으로 줄바꿈됩니다. 화면 밖으로 흘러 안 보이던 문제가 해결됐습니다.
  • 알림 종 숫자가 빠르게 갱신됩니다.
  • 사이드바 아래쪽 「인플루언서 화면」·「로그아웃」 메뉴가 위쪽 메뉴와 같이 스크롤되도록 합쳐졌습니다.

인플루언서 앱 — 홈

  • 인기 캠페인 10개와 「더보기」 버튼이 새로 생겼습니다. 가장 활발한 캠페인 10개를 홈에서 바로 보여드리고, 「더보기」를 누르면 전체 캠페인 페이지로 이동합니다.
  • 3개씩 가로로 펼친 카드가 화면 밖으로 흐르지 않도록 정리됐습니다. (일본어 화면 기준)

인플루언서 앱 — 캠페인 페이지 (목록)

  • 화면을 위로 당기면 새로고침됩니다. 페이지 맨 위에서 화면을 아래로 당기면 새로고침 아이콘이 뜨고, 더 당기면 새로고침이 실행됩니다. 모바일에서 익숙한 동작이라 사용자에게 친숙합니다.
  • 스크롤을 내리면 필터 영역만 자동으로 접힙니다. 화면을 더 넓게 쓸 수 있고, 스크롤을 다시 위로 올리면 필터가 다시 보입니다. 제목과 모집유형 탭은 항상 고정되어 있습니다.
  • 검색은 돋보기 아이콘을 눌러 엽니다. 돋보기를 누르면 제목 자리에 검색창이 나타나고, Enter 또는 돋보기를 다시 누르면 검색됩니다.
  • 필터를 바꾸면 목록이 자동으로 맨 위로 올라갑니다.

인플루언서 앱 — 캠페인 상세

  • 상품 정보가 표 형태로 깔끔하게 정리됐습니다. 리뷰어 캠페인의 「ペイバック(페이백)」 표현도 더 명확해졌습니다.
  • 섹션 사이 구분이 산뜻해졌습니다. 두꺼운 회색 막대였던 구분선이 가는 점선으로 바뀌었고, 콘텐츠가 양옆 끝까지 시원하게 보이도록 안쪽 여백을 정리했습니다.

인플루언서 앱 — 마이페이지·신청

  • 신청할 때 「한자 이름」과 「가나 이름」이 둘 다 등록되어 있어야 합니다. 둘 중 하나라도 비어 있으면 신청이 차단되고 안내 메시지가 뜹니다. 일본 배송 라벨에 둘 다 필요해서 추가했습니다.
  • 마이페이지에 「未登録(미등록)」 빨간 배지가 뜹니다. 한자 이름이나 가나 이름이 비어 있을 때 표시됩니다. 인플루언서가 직접 채우라고 안내하시면 됩니다.
  • 마이페이지에 들어갈 때마다 프로필을 다시 불러옵니다. 다른 기기나 다른 탭에서 변경한 내용이 즉시 반영됩니다.
  • 로고를 누르면 홈으로 새로고침해서 이동합니다.

한국어 번역 정정 (한국어로 보는 인플루언서 한정)

  • 캠페인 모집유형 라벨 「Reviewer」를 「리뷰어」로 수정
  • 「¥1,000원」처럼 잘못 적혔던 표현을 「¥1,000엔」으로 수정
  • 일부 플로팅 버튼과 결제 페이백 라벨에 한국어 번역 추가

운영팀 자주 묻는 질문

1. 「일시정지」였던 캠페인은 어떻게 됐나요?

자동으로 「종료」 상태로 바뀌었습니다. 다시 모집을 켜고 싶으면 드롭다운에서 「모집중」으로 바꾸시면 됩니다.

2. 「노출마감」은 어떻게 들어가나요?

「종료」 상태에서 게시 기한이 지나면 시스템이 자동으로 「노출마감」으로 바꿉니다. 노출마감된 캠페인은 인플루언서 화면에서 완전히 안 보이고, 관리자 화면에서는 회색 점선으로 표시됩니다.

3. 결과물 엑셀 줄 수가 늘어났는데 정상인가요?

네, 정상입니다. 신청 1개당 1줄로 펼쳐서 영수증과 리뷰 이미지, 게시물 주소를 한 줄에서 같이 볼 수 있도록 새로 만들었습니다.

4. 인플루언서가 「이름이 등록되어 있지 않다」고 문의하면?

마이페이지 → 기본정보에서 「한자 이름」과 「가나 이름」 둘 다 채우라고 안내하시면 됩니다.

5. 광고주 시딩 폼의 송금 수수료는 어떻게 입력하나요?

직접 입력하지 않으셔도 됩니다. 자동으로 0원으로 채워집니다.

궁금한 점이나 작동이 이상한 부분이 있으시면 개발팀에 알려주시기 바랍니다.
