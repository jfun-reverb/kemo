브랜드 오리엔시트 발급·작성·발행 기능이 열렸습니다

이제 브랜드(광고주)가 로그인 없이 받은 링크로 직접 캠페인 콘텐츠(오리엔시트)를 작성하고, 관리자가 그 내용을 캠페인 등록 폼에 자동으로 옮겨 발행할 수 있습니다. 기존 구글시트 오리엔시트를 대체하는 흐름입니다.

전체 흐름은 3단계입니다.

  1. 발급 — 관리자가 브랜드에게 작성 링크를 만들어 메일로 보냅니다.
  2. 작성 — 브랜드가 받은 링크로 들어가 한국어로 내용을 채워 제출합니다.
  3. 발행 — 관리자가 제출된 내용을 캠페인으로 자동 채워 발행합니다.

1. 작성 링크 발급하기 (관리자)

발급은 아래 세 곳 어디서든 시작할 수 있습니다.

  • 왼쪽 메뉴 「오리엔시트 현황」 → 새 발급 버튼
  • 「서베이 신청 목록」에서 해당 신청 행의 더보기(⋯) → 「시스템 오리엔시트 발급」
  • 「브랜드 관리」에서 브랜드 상세 → 「오리엔시트 발급」

발급 시 브랜드만 선택하면 작성 형식(가구매·리뷰어·시딩)은 브랜드가 작성 화면에서 직접 고릅니다. 신청과 연결해 발급하면 브랜드명·제품명 등 기본 정보가 미리 채워집니다.

2. 발급 후 작성 링크 메일 보내기 (관리자)

발급하면 결과 화면에서 작성 링크가 만들어지고, 브랜드 담당자에게 메일이 자동 발송됩니다.

  • 받는 사람은 대표 담당자가 기본으로 잡히며, 드롭다운에서 다른 담당자를 고르거나 이메일을 직접 입력할 수 있습니다.
  • 직접 입력한 이메일은 발송 성공 후 그 브랜드의 담당자 정보로 저장됩니다.
  • 이미 보낸 수신자에게는 발송 버튼이 잠겨 같은 사람에게 중복 발송되는 것을 막습니다. 발송한 시각도 함께 표시됩니다.
  • 모달을 닫았어도 「오리엔시트 현황」 목록의 행 「메일」 버튼으로 다시 들어가 재발송할 수 있습니다.

3. 제출된 내용으로 캠페인 발행하기 (관리자)

브랜드가 작성을 마치면 「오리엔시트 현황」에서 상태가 바뀝니다. 상세를 열어 내용을 확인한 뒤 카드별 「이 카드로 발행」을 누르면 캠페인 등록 폼에 내용이 자동으로 채워집니다.

  • 브랜드는 한국어로만 작성하므로, 일본어 칸(제품명·제목 등)은 비어 있습니다. 관리자가 일본어를 채워야 발행됩니다.
  • 한 장에 여러 모집(카드)이 있으면 카드마다 캠페인이 한 건씩 만들어집니다.

현황 화면에서 한눈에 보기

  • 목록 상단의 상태 탭으로 작성중·제출됨 등을 걸러 볼 수 있습니다.
  • 「모집 형식」 칸에 가구매·리뷰어·시딩이 몇 건인지 표시됩니다.
  • 일부 카드만 발행된 시트에는 부분 발행 표시가 붙습니다.
  • 발급 목록에서 시트 삭제도 가능합니다. 단, 연결된 캠페인에 신청자가 한 명이라도 있으면 삭제가 막힙니다.

권한별로 할 수 있는 일

관리자 권한은 최고 관리자 > 광고 관리자 > 매니저 3단계입니다. 오리엔시트에서는 대부분의 작업을 매니저를 포함한 모든 관리자가 할 수 있고, 신중함이 필요한 두 가지만 광고 관리자 이상으로 제한됩니다.

매니저 포함 모든 관리자가 할 수 있는 일:

  • 작성 링크 발급 및 브랜드에게 메일 발송 (관리자로 로그인한 본인만 발송됩니다)
  • 제출된 오리엔시트 내용 조회
  • 정상 발행 (일본어 보완을 마친 캠페인 발행)
  • 아직 발행하지 않은 시트의 삭제

광고 관리자 이상만 할 수 있는 일:

  • 긴급 발행 — 일본어 콘텐츠 가이드를 비운 채 발행하는 경우입니다. 매니저는 막히고, 광고 관리자 이상은 사유를 입력해야 발행됩니다(사유는 기록에 남습니다).
  • 이미 발행된 캠페인과 연결된 시트의 삭제 — 단, 그 캠페인에 신청자가 한 명이라도 있으면 누구도 삭제할 수 없습니다.

영향 범위: 관리자 페이지 한정. 운영서버에 배포 완료됐습니다.
