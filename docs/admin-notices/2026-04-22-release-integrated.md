### 본문 (Markdown — Quill 에디터에 붙여넣기)

안녕하세요. 오늘(2026-04-22) 운영 관리자 페이지에 **큰 기능 묶음**이 배포됐습니다. 아래 8가지 주요 변경사항을 확인해주세요.

---

**① 🆕 공지사항 메뉴가 생겼습니다**

사이드바 **맨 위** `공지사항` 메뉴에서 팀 전체에 알림을 공유할 수 있습니다.

- **미읽음 배지**: 메뉴 옆에 내가 아직 안 본 공지 건수가 표시됩니다 (9+면 `9+`)
- **로그인할 때 팝업**: 관리자 로그인 직후 미읽음 공지가 있으면 모달로 알려줍니다
- **4가지 카테고리**: 시스템 업데이트 · 릴리스 · 경고 · 일반
- **상단 고정(핀)**: 중요한 공지는 핀으로 위에 고정 가능
- **대시보드 카드**: 대시보드에서도 최근 3건 공지 확인 가능

이 공지 자체도 **바로 공지사항 메뉴에 등록**해서 팀 전체에 공유해보세요.

---

**② 🆕 인플루언서 관리 화면 대폭 개편**

인플루언서 목록 → 행 클릭 → 상세 모달에서 **인증·위반·블랙리스트**를 한 화면에서 관리합니다.

- **이름 옆 배지**: 블랙리스트는 빨간 `블랙` 단독, 아니면 `인증` + `위반 N회`(회색) 함께 표시
- **상태 관리 카드**: 인증 토글, 위반 등록, 블랙리스트 등록/해제 버튼을 한곳에 모음
- **관리자 이력 타임라인**: 사유별 누적 횟수 pill + 시간순 기록
- **증빙 파일 첨부**: 위반/블랙 등록 시 이미지·PDF 업로드 가능 (10MB 이하). 작은 썸네일 클릭하면 확대해서 볼 수 있어요
- **위반 기록 수정**: 위반 행 옆 연필 아이콘 → 사유·메모·증빙 나중에라도 수정 가능
- **목록 필터 개편**: 상단에 채널·인증·위반 드롭다운 3종과 통합 검색 추가

---

**③ 🆕 브랜드 서베이 상태가 9단계로 확장됐습니다**

기존 5단계(new → reviewing → quoted → paid → done)에서 더 세분화됐습니다.

| 순서 | 상태 | 의미 |
|---|---|---|
| 1 | new | 방금 접수 |
| 2 | reviewing | 검토 시작 |
| 3 | **schedule_sent** (신규) | 일정 안내 메일 발송 |
| 4 | quoted | 견적서 발송 완료 |
| 5 | **orient_sheet_sent** (신규) | 오리엔시트 전달 |
| 6 | **campaign_registered** (신규) | 캠페인 등록 완료 |
| 7 | paid | 입금 확인 |
| 8 | done (**최종완료**) | 운영 종료 |
| — | rejected | 거절 (어느 단계에서든 전환 가능) |

"완료" 라벨은 **"최종완료"** 로 바뀌었습니다. 대시보드 깔때기 차트도 9단계로 업데이트됐어요.

캠페인 목록의 각 행 **⋮ 더보기 메뉴** 및 브랜드 서베이 페인 헤더에서 엑셀을 다운받을 수 있습니다.

- **브랜드 서베이 엑셀** (신규): 접수번호·폼타입·업체명·담당자·연락처·예상견적·상태 등

**⑤ 🆕 필터가 훨씬 편해졌어요 (신청 관리 · 결과물 관리)**

- **캠페인을 여러 개 동시에** 선택해서 볼 수 있는 **다중 선택 드롭다운**으로 바뀌었습니다
- **타입 cascade**: 캠페인을 고르면 해당 캠페인에 맞는 타입(리뷰어/기프팅/방문형)만 자동으로 옵션에 남아 선택 실수가 줄어듭니다
- 필터 위치를 **맨 왼쪽**으로 통일

**⑥ 🆕 신청자 목록에 SNS 4채널이 모두 보여요**

신청자 행에 Instagram · TikTok · X · YouTube **4개 채널의 아이디와 팔로워 수**가 모두 표시됩니다. 멀티채널 캠페인(예: IG+X 동시 모집)에서 어느 채널 기준으로 자격이 맞는지 한눈에 판단 가능.

**⑦ 🆕 전화번호가 깔끔하게 표시됩니다**

한국 번호(`010-1234-5678`), 일본 번호(`090-1234-5678`, `+81-90-...`) 모두 자동으로 하이픈 포맷이 적용됩니다. 인플루언서 상세 모달, 브랜드 서베이 리스트·상세에 반영.

---

**⑧ 🆕 인플루언서 앱 — 주의사항 동의 단계 추가**

캠페인 등록 시 **주의사항** 항목을 설정해두면, 인플루언서가 신청 모달 상단에서 빨간 박스로 확인하고 **"全ての注意事項を確認しました"** 체크박스에 동의해야만 신청 완료가 됩니다.

- 동의 시점(`caution_agreed_at`)과 당시 주의사항 내용 스냅샷이 함께 저장됩니다 — 나중에 주의사항을 수정해도 **신청 당시의 내용이 그대로 보존**됩니다
- 기준데이터 관리 → 주의사항(`caution`) 탭에서 공통 주의사항 풀을 관리할 수 있습니다
