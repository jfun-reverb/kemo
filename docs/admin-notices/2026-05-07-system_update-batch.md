## 관리자 페이지

### 캠페인 관리
- **캠페인 상태 5단계 재설계** — `paused`(일시정지) 제거 + `expired`(노출마감) 신규. 5단계 = `draft / scheduled / active / closed / expired`. 노출 그룹(scheduled / active / closed) 컬러 배지(파랑/초록/핑크) + 비노출 그룹(draft / expired) 회색·점선. `closed` 캠페인이 `post_deadline`(게시 기한) 경과 시 `expired`로 자동 전환되어 인플루언서 화면에서 완전히 사라짐 (migration 097)
- **캠페인 폼 3단 날짜 레이아웃** — 모집 기간 / 구매·방문 기간 / 결과물 제출 마감을 3열로 배치. 모집 종료 시 결과물 마감 +14일 자동 제안이 구매·방문 picker 옆으로 이동. `closed` 캠페인 날짜 수정 시 잘못된 차단 모달이 떴던 문제 해결

### 결과물 관리
- **결과물 엑셀 재설계 — 신청 단위 18컬럼** — 기존 결과물 단위 행 → 신청 단위로 펼쳐 영수증·리뷰 이미지·게시물 URL을 한눈에 비교. 인플루언서 계정 이메일 컬럼 추가. monitor STEP 2 리뷰 이미지(`review_image`) 결과물도 엑셀에 포함
- **검수 모달에 리뷰 이미지 렌더링** — 영수증과 리뷰 이미지를 모달 한 화면에서 같이 검수 가능

### 광고주 신청 (브랜드 서베이)
- **시딩 송금 수수료 자동 0원** — 시딩 폼은 송금 수수료 항상 0원으로 자동 채움 (이전: 수동 입력 필요했음). 관리자 직접 등록 모달도 동일 적용 (migration 098 / 099)

### 관리자 일반 UI
- **목록 페인 4종 가로 스크롤 + 첫 컬럼 좌측 고정** — 캠페인 관리 / 신청 관리 / 결과물 관리 / 캠페인별 신청자 페인이 노트북 화면에서 좁을 때 가로 스크롤 + 첫 번째 컬럼(주로 썸네일·제목)은 항상 왼쪽 고정
- **목록 옵션별 (N) 건수 표시** — 캠페인 / 신청 / 결과물 / 광고주 신청 페인 필터 드롭다운 옵션 옆에 해당 건수 표시. deliverables 필터에는 sub-label도 추가, 기본값 「미제출 포함 ON」 + 「전체」 토글 표준화
- **모달 저장 후 뒤 페인 자동 새로고침** — `refreshPane()` 헬퍼 도입. 모달 저장 후 뒤의 목록·집계가 이전 상태로 남아 있던 문제 해결
- **사이드바 메뉴 클릭 시 페이지 전체 새로고침** — 페인 간 이동 시 잔여 모달·필터·캐시 차단. 매번 깨끗한 상태로 시작
- **multi-select 드롭다운 라벨 줄바꿈** — 긴 캠페인 제목도 화면 안에 안전하게 표시
- **알림 배지 즉시 갱신** — 폴링 시작 시 배지 수치를 즉시 갱신
- **사이드바 푸터 → 스크롤 영역 통합** — 인플루언서 화면·로그아웃 메뉴가 사이드바 메뉴와 같은 스크롤 안으로 통합되어 일관된 메뉴 경험
- **한국어 오탈자 자동 점검 패턴 추가** — 배포 전 hook이 자주 틀리는 단어를 자동 검출 (운영자 직접 작업에는 변화 없음)

## 인플루언서 앱

### 홈
- **인기 캠페인 top 10 + 더보기 버튼** — 홈에서 가장 활발한 10개를 보여주고 「더보기」로 전체 캠페인 페이지 이동
- **3단 카드 레이아웃 안정화** — 일본어 로케일에서 카드가 컨테이너 밖으로 흘렀던 문제 수정

### 캠페인 페이지 (목록)
- **pull-to-refresh** — 페이지 최상단에서 아래로 당기면 새로고침 아이콘이 나타나고 더 당기면 활성화. 트리거 임계값 90px (휴대폰 스크롤 자연스러움 우선)
- **스크롤 다운 시 필터만 숨김** — 스크롤 내릴 때 필터 영역만 접히고, 스크롤 올리면 다시 노출. 제목·모집유형 탭은 항상 고정 (화면 활용도 ↑)
- **검색 아이콘 토글** — 돋보기 클릭 시 제목과 검색 입력란이 같은 자리에서 swap. Enter 또는 돋보기 다시 클릭으로 검색
- **필터·탭 변경 시 자동 스크롤 top** — 필터를 바꾸면 목록이 자동으로 맨 위로 이동

### 캠페인 상세
- **상품 정보 표 재구성** + monitor(리뷰어) 캠페인의 「ペイバック」 표현 명확화 + 섹션 색상 톤 정리
- **좌우 여백·구분선 정리** — 브랜드명·모집타입·제목 영역과 참여방법·캠페인 설명·주의사항 영역 좌우 안쪽 여백 제거 (외부 여백은 유지). 섹션 사이의 8px 회색 구분 바를 1px 점선으로 교체

### 마이페이지·신청
- **신청 모달 한자/가나 이름 가드** — 캠페인 신청 시 한자 이름·가나 이름이 둘 다 등록되어 있어야 신청 가능. 둘 중 하나라도 비어 있으면 신청 차단 + 안내 메시지 (배송 라벨 누락 사고 방지)
- **마이페이지 「未登録」 배지** — 한자/가나 이름이 비어 있으면 마이페이지 메뉴에 빨간 「未登録」 배지 노출
- **마이페이지 진입마다 프로필 재페치** — 다른 탭·디바이스에서 변경한 내용이 즉시 반영
- **로고 클릭 시 홈 새로고침** — 어느 페이지에서든 로고 클릭으로 깨끗한 홈으로 복귀

## 운영팀 자주 묻는 질문 (FAQ)

1. **「일시정지(`paused`)」 상태였던 캠페인은 어떻게 됐나요?**
   migration 097이 자동으로 `closed`로 전환했습니다. 다시 활성으로 되돌리려면 드롭다운에서 `active`로 변경하시면 됩니다.

2. **「노출마감(`expired`)」은 어떻게 들어가는 건가요?**
   `closed` 상태에서 `post_deadline`(게시 기한) 경과 시 자동 전환됩니다. expired 캠페인은 인플루언서 화면에서 완전히 사라지고 관리자 화면에서는 회색·점선으로 표시됩니다.

3. **결과물 엑셀이 행이 늘어났는데 정상인가요?**
   네, 신청 단위로 펼쳐 영수증 / 리뷰 이미지 / 게시물 URL을 한눈에 비교할 수 있도록 재구성한 결과입니다. 컬럼은 18개 + 인플루언서 이메일 컬럼이 추가됐습니다.

4. **인플루언서가 「이름 미등록 알림」 문의를 하면?**
   마이페이지 → 기본정보에서 한자 이름·가나 이름 둘 다 채우라고 안내하시면 됩니다.

5. **광고주 시딩 폼의 송금 수수료는 어떻게 입력하나요?**
   자동 0원으로 채워집니다. 별도 입력하지 않아도 됩니다.
