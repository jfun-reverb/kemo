# 메시지 기능 도입 — 30일 사전 통지 문안

> 응모건 메시지 기능 운영 배포에 따른 약관·개인정보처리방침 개정 사전 통지.
> **통지 기간 30일**(자사 약관 부칙 "수집항목 변경 30일 전 공지" 준수).
> 인플루언서 대상이므로 **일본어가 실제 발송본**, 한국어는 사용자 검토용 병기.
> **시행일 `{시행일}`** = 통지 게시일 + 30일 (게시 확정 시 확정).

## ① 회원 메일

### 제목 (일본어 발송본)
```
【REVERB JP】「メッセージ」機能の追加と、規約・プライバシーポリシー改定のお知らせ
```
（한국어 검토용: 【REVERB JP】「메시지」 기능 추가 및 약관·개인정보처리방침 개정 안내）

### 본문 (일본어 발송본)
```
REVERB JP をご利用いただきありがとうございます。

このたび、応募されたキャンペーンごとに運営チームへ直接お問い合わせができる
「メッセージ」機能を追加します。あわせて、利用規約とプライバシーポリシーを
一部改定いたします。

■ 何が変わりますか
1. 応募履歴の各キャンペーンから、運営チームにメッセージ（文章・画像）を
   送れるようになります。
2. これまで LINE でいただいていた個別のお問い合わせを、アプリ内でも
   受け付けます。

■ 新しく集める情報
・メッセージの本文（テキスト）と、添付した画像
・利用目的：応募件に関する運営チームへのお問い合わせ・返信・確認のため
・保管期間：応募が終了してから1年（退会された場合はその時点で削除）

■ メッセージを見られる人
・送信したご本人と、REVERB JP の運営チーム担当者だけが見られます。
・他の会員のメッセージは見ることができません。
・担当者がメッセージを開いたり画像を保存したりした記録は、安全のために
  すべて残されます。

■ 同意について
メッセージはサービス運営のための連絡手段のため、別途の同意操作は不要です。
施行日以降も続けてご利用いただくことで、改定内容にご同意いただいたものとします。
ご同意いただけない場合は、メッセージ機能をご利用にならないか、退会の手続きが
可能です。

■ 施行日
{시행일}

改定後の利用規約・プライバシーポリシーの全文は、アプリ下部の
「利用規約」「個人情報処理方針」からご確認いただけます。

ご不明な点は、公式LINE @reverb.jp までお問い合わせください。
```

## 한국어 검토용 전문 (발송 안 함, 사용자 확인용)

### 메일 본문 (한국어)
```
REVERB JP를 이용해 주셔서 감사합니다.

이번에 응모하신 캠페인별로 운영팀에 직접 문의할 수 있는 「메시지」 기능을
추가합니다. 함께 이용약관과 개인정보처리방침을 일부 개정합니다.

■ 무엇이 바뀌나요
1. 응모이력의 각 캠페인에서 운영팀에 메시지(글·이미지)를 보낼 수 있습니다.
2. 그동안 LINE으로 받던 개별 문의를 앱 안에서도 받습니다.

■ 새로 수집하는 정보
· 메시지 본문(텍스트)과 첨부 이미지
· 이용 목적: 응모건 관련 운영팀 문의·답변·확인
· 보관 기간: 응모 종료 후 1년 (탈퇴 시 그 시점에 삭제)

■ 메시지를 볼 수 있는 사람
· 보낸 본인과 REVERB JP 운영팀 담당자만 볼 수 있습니다.
· 다른 회원의 메시지는 볼 수 없습니다.
· 담당자가 메시지를 열람·다운로드한 기록은 안전을 위해 모두 남습니다.

■ 동의에 대하여
메시지는 서비스 운영을 위한 연락 수단이므로 별도 동의 절차는 필요 없습니다.
시행일 이후에도 계속 이용하시면 개정 내용에 동의하신 것으로 봅니다.
동의하지 않으시면 메시지 기능을 이용하지 않거나 탈퇴하실 수 있습니다.

■ 시행일
{시행일}

개정된 약관·개인정보처리방침 전문은 앱 하단 「이용약관」「개인정보처리방침」에서
확인하실 수 있습니다.

문의는 공식 LINE @reverb.jp 으로 부탁드립니다.
```
