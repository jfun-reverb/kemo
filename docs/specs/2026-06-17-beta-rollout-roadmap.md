## 3단계 로드맵

### 베타 1차 — 인플루언서 운영 핵심 (~10주, 2026년 8월 말 목표)
| 기능 | 기간 | 비고 |
|---|---|---|
| 연령 정책 마무리(PR5) | W1 | PR1~4 dev 완료. 약관·시행만 |
| 인플루언서 정산 관리 | W2~4 | 정산 리스트·내역(페이팔 송금)·권한·인플 조회 |
| 포인트 제도 | W4~6 | 정산과 **연속**(결과승인→지급 흐름·정산내역 화면 공유로 절감) |

- **약관**: 위 통합 통지분이 이 시점엔 시행 완료.
- **리스크**: 금전 정확성·세무(포인트 현금화/아마존 쿠폰). 실데이터 검증 비용이 개발보다 클 수 있음.

### 베타 2차 — 브랜드 포털 (~14주, 2026년 9월 말 목표)
| 기능 | 기간 | 비고 |
|---|---|---|
| 브랜드 페이지 | W6~11(1차와 일부 병행 기획) | 로그인·회원관리·권한·캠페인 리스트/현황/대금 |
| 오리엔시트 | W10~13 | 작성 URL·클라이언트 페이지(리뷰어/시딩)·관리자 관리·캠페인 생성 연동 |
- **약관**: 브랜드 계약·개인정보 별도 개정(브랜드사 대상).
- **리스크 (최대)**: **세 번째 사용자 유형(브랜드사)** 추가 = 로그인·인증·행 단위 보안 정책(RLS)·보안 전면 신규. 가장 길고 불확실. 일정이 여기서 가장 크게 흔들림. → **베타 1차 개발 중 미리 상세 기획**해 리스크 조기 발견 권장.
- sales(브랜드 서베이) 도메인·`brand_applications`·오리엔시트 컬럼 일부 기반 재사용 가능.
