## 0. 중대 변경 여부 판단 (`.claude/rules/policy.md`)
- B·C·D = 기존 의무(금지행위·콘텐츠 게시·리워드)의 **명확화**. 신규 수집·제3자 제공·국외 이전 추가 **없음** → **중대 변경 아님**.
- F = 기존에 보관하던 위반 기록의 **보관 기간 명시**(투명성 향상). 신규 수집 아님 → **중대 변경 아님**.
- 따라서 **시행일 7일 전 앱 내 공지로 충분**(미성년 정책처럼 재동의·전체 메일 통지까지는 불필요). 한·일 동시 개정 + 상단 최종 갱신일·시행일 갱신.

## B. 허위·도용 리뷰 금지 — 약관 제16조(회원의 금지 행위)에 항목 추가

현재 제16조는 1~8번. **9·10번 신설.**

**한국어 (`TERMS_kr.md` 제16조):**
```
9. 타인의 게시물·리뷰를 도용하거나 편집을 통해 허위 후기를 작성하는 행위
10. 사실과 다른 비방성 후기 또는 악의적 리뷰로 회사·브랜드에 손해를 끼치는 행위
```

**일본어 (`TERMS_ja.md` 第16条):**
```
9. 他人の投稿・レビューを盗用する、または編集により虚偽のレビューを作成する行為
10. 事実と異なる誹謗的なレビュー、または悪意あるレビューにより当社・ブランドに損害を与える行為
```

## C. 광고 표기 정정 요구권 — 약관 제11조(콘텐츠 게시 의무)에 항 추가

현재 제11조는 1~4번. **5번 신설** (일본 스테마 규제 대응 강화).

**한국어 (`TERMS_kr.md` 제11조):**
```
5. 회사 또는 브랜드는 광고 표기(지정 해시태그) 누락, 가이드라인 위반, 부적절한 내용이 포함된 콘텐츠에 대해 정정을 요구할 수 있습니다. 회원이 정해진 기한 내에 정정 요구에 응하지 않을 경우, 회사는 제14조의 노쇼 또는 제재 규정을 적용할 수 있습니다.
```

**일본어 (`TERMS_ja.md` 第11条):**
```
5. 当社またはブランドは、広告表記（指定ハッシュタグ）の欠落、ガイドライン違反、または不適切な内容を含むコンテンツについて訂正を要求することができます。会員が定められた期限内に訂正要求に応じない場合、当社は第14条のノーショーまたは制裁規定を適用することができます。
```

## D. 리워드 부정 수령 환수 — 약관 제13조(리워드 지급)에 항 추가

현재 제13조는 1~3번. **4번 신설.**

**한국어 (`TERMS_kr.md` 제13조):**
```
4. 회원이 허위·부정한 방법(중복 수령, 위·변조 영수증, 자격 위조 등)으로 리워드를 수령한 경우, 회사는 해당 리워드를 환수하고 향후 리워드 지급을 제한할 수 있습니다.
```

**일본어 (`TERMS_ja.md` 第13条):**
```
4. 会員が虚偽・不正な方法（重複受領、偽造・改ざんレシート、資格の偽装等）により報酬を受領した場合、当社は当該報酬を回収し、今後の報酬支払いを制限することができます。
```

## F. 위반·부정이용 기록 보관 기간 명시 — 개인정보처리방침 §6.1 보유기간 표에 행 추가

현재 `influencer_flags`(위반·블랙리스트·노쇼 마킹)를 실제로 보관하나 정책에 보관 기간 명시가 없음. **표에 행 추가.**

**한국어 (`PRIVACY_kr.md` §6.1 표):**
```
| 부정 이용·위반 기록(노쇼, 블랙리스트, 위반 마킹) | 3년 | 부정 이용 방지·분쟁 대응 |
```

**일본어 (`PRIVACY_ja.md` §6.1 表):**
```
| 不正利用・違反記録（ノーショー、ブラックリスト、違反マーキング） | 3年 | 不正利用防止・紛争対応 |
```

> ⚠️ **보관 기간 3년은 실제 운영 보관과 일치시킬 것.** 현재 `influencer_flags` 행을 영구 보관한다면, 정책 명시값(3년)에 맞춰 3년 경과 분 정리 로직을 두거나, 정책값을 실제에 맞게 조정. 개발 세션이 실제 보관 정책을 확인 후 숫자 확정.

## 1. 작업 절차 (개발 세션)
1. `TERMS_kr.md` / `TERMS_ja.md` 제11·13·16조에 위 문구 삽입 (번호 이어서)
2. `PRIVACY_kr.md` / `PRIVACY_ja.md` §6.1 표에 행 추가
3. 네 파일 모두 상단 **최종 갱신일** + 하단 **부칙 공고일·시행일** 갱신 (시행일은 공고 7일 후)
4. **한국어 오탈자 자체 점검** + 일본어 문구 자연스러움 확인
5. F의 보관 기간(3년)을 실제 `influencer_flags` 운영 보관과 정합화

## 2. 영향 / 리스크
- 코드·DB 변경 **없음**(문서만). 단 F는 보관 기간 명시 ↔ 실제 보관 정합성 확인 필요.
- 중대 변경 아님 → 7일 전 앱 공지로 충분. 별도 재동의·메일 불필요.
- 미성년 정책(별도 사양서)과 **운영 배포를 묶으면** 통지를 한 번에 처리 가능(단 미성년 정책은 중대 변경이라 메일 통지 필요 — 묶을 경우 메일에 두 변경 모두 안내).
