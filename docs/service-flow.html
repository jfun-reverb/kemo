3캠페인 상태 라이프사이클

캠페인의 5가지 상태(draft→scheduled→active→paused→closed)와 전환 조건, 노출 규칙
stateDiagram-v2
  [*] --> draft: 캠페인 등록

  draft --> scheduled: 관리자 상태 변경
  draft --> active: 관리자 상태 변경 (deadline 유효시)

  scheduled --> active: 관리자 상태 변경
  scheduled --> draft: 관리자 상태 변경

  active --> paused: 관리자 일시정지
  active --> closed: 관리자 수동 종료
  active --> closed: deadline 경과 (자동)

  paused --> active: 관리자 재개 (deadline 미경과시만)
  paused --> closed: 관리자 종료

  closed --> draft: 관리자 재등록

  note right of draft
    준비중 — 비공개
    목록 미노출
  end note
  note right of scheduled
    공개예정 — 목록 노출
    신청 불가
  end note
  note right of active
    모집중 — 목록 노출
    신청 가능
    deadline 경과 시 자동 closed
  end note
  note right of paused
    일시정지 — 목록 미노출
    deadline 경과시 active 전환 차단
  end note
  note right of closed
    모집마감
    post_deadline 남으면 → 모집마감으로 목록 유지
    post_deadline 경과 → 완전 비노출
  end note
