# 영수증 및 활동 관리 기획서 (초안)

> ⚠️ **Superseded (2026-04-15)**: 이 문서는 영수증 한정 초안입니다. 게시물 URL 제출(기프팅/방문형)까지 통합한 최신 기획은 [`plans/2026-04-15_deliverables-management.md`](plans/2026-04-15_deliverables-management.md) 참조.

## 1. 현재 구현 상태

### 인플루언서 (완료)
- 승인된 캠페인 상세 → "활동관리" 버튼 → 활동관리 페이지
- 마이페이지 응모이력 → 승인된 캠페인 클릭 → 활동관리 페이지
- 영수증 등록: 이미지 업로드 + 구매일 + 금액
- 등록된 영수증 리스트 표시

### 관리자 (미구현)
- 영수증 조회/확인/반려 기능 없음
- 활동 완료 처리 기능 없음

## 2. 활동 플로우 (안)

### 기본 플로우

캠페인 신청(인플루언서) → 승인(관리자) → 영수증 등록(인플루언서) → 영수증 확인/반려(관리자) → 활동 완료(관리자)

### 반려 시

영수증 반려(관리자) → 영수증 재제출(인플루언서) → 영수증 확인/반려(관리자)

### 향후 확장 (검토 필요)

영수증 확인(관리자) → SNS 리뷰 등록(인플루언서) → 리뷰 확인(관리자) → 활동 완료(관리자) → 정산

### 활동 상태값

| 상태 | 설명 |
|------|------|
| 대기 | 승인됨, 활동 대기 중 |
| 영수증 제출 | 인플루언서가 영수증을 등록한 상태 |
| 영수증 확인 | 관리자가 영수증을 확인한 상태 |
| 영수증 반려 | 관리자가 영수증을 반려한 상태 (재제출 필요) |
| 리뷰 제출 | 인플루언서가 SNS 리뷰를 등록한 상태 (향후) |
| 활동 완료 | 관리자가 활동 완료를 처리한 상태 |

> **결정 필요**: SNS 리뷰 등록 단계를 포함할지, 영수증만으로 완료할지

## 3. 관리자 기능 (구현 필요)

### 3-1. 영수증 확인 위치

| 방안 | 설명 | 장점 | 단점 |
|------|------|------|------|
| A. 신청자 상세 모달 | 인플루언서 클릭 시 모달에 영수증 탭 추가 | 기존 화면 활용, 개발 빠름 | 전체 영수증 현황 파악 어려움 |
| B. 캠페인별 신청자 목록 | 승인된 신청자 옆에 영수증 상태 표시 + 클릭 시 확인 | 캠페인 단위 관리 용이 | 전체 현황은 별도 필요 |
| C. 별도 영수증 관리 페이지 | 사이드바에 메뉴 추가, 전체 영수증 조회 | 전체 현황 파악 용이, 필터/검색 | 개발 공수 큼 |
| D. B + C 조합 | 캠페인별 + 전체 관리 모두 | 가장 유연 | 개발 공수 가장 큼 |

### 3-2. 영수증 확인/반려 처리

- **확인**: 영수증 이미지 + 구매일 + 금액 검토 → "확인" 버튼
- **반려**: 반려 사유 입력 → 인플루언서에게 재제출 요청
- **반려 사유 예시**: 이미지 불선명, 날짜 불일치, 금액 불일치, 타 상품 영수증

### 3-3. 활동 완료 처리

- 영수증 확인 완료 후 → "활동 완료" 버튼
- 완료된 활동은 대시보드 통계에 반영

## 4. 인플루언서 기능 (추가 필요)

### 4-1. 영수증 상태 표시
- 활동관리 페이지에서 영수증 상태 표시: 제출 → 확인중 → 확인완료 / 반려
- 반려 시 사유 표시 + 재제출 가능

- 관리자 확인 후 활동 완료
